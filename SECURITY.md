# Security Policy

## Reportando uma Vulnerabilidade

Se você descobrir uma vulnerabilidade de segurança no lojinha-agro, pedimos que por favor nos notifique imediatamente. Levamos todas as vulnerabilidades de segurança muito a sério.

**Não crie uma issue pública** para reportar a vulnerabilidade.

Em vez disso, por favor:

1. Descreva o problema em detalhes
2. Inclua passos para reproduzir
3. Inclua a versão afetada
4. Sugira uma possível solução se tiver uma

## Versões Suportadas

| Versão | Suportada          |
| ------ | ------------------ |
| 1.0.x  | :white_check_mark: |
| < 1.0  | :x:                |

## Política de Divulgação

Quando recebemos um relatório de segurança, nós:

- Confirmamos o recebimento do relatório
- Determinamos a gravidade do problema
- Desenvolvemos e testamos uma correção
- Preparamos uma atualização para todas as versões afetadas ainda suportadas
- Publicamos a correção
